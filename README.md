# Sistema BNCC com IA

## 🚀 Como usar

1. Baixe ou clone o projeto
2. Abra no VS Code
3. Coloque sua chave da API Gemini no app.js
4. Execute com Live Server

## 🌐 Deploy

Pode usar:
- GitHub Pages
- Vercel
- Netlify

## 🤖 Funcionalidades

- Busca de habilidades BNCC
- Seleção rápida
- Geração automática de plano com IA
- Estrutura pedagógica completa

## 🔒 Observação

Nunca publique sua chave da API em repositórios públicos.
